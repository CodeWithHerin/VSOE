# Venice Simplon-Orient-Express (VSOE)

An ultra-luxury web experience for the Venice Simplon-Orient-Express, aiming to match and exceed the official digital presence.

## Features
- **Official Aesthetic**: Dark "Midnight" theme with "Cream" and "Gold" accents.
- **Interactive Map**: High-energy, minimalist route map with video previews.
- **Cinematic Experience**: Video backgrounds and smooth, liquid scroll animations.
- **Dynamic Navigation**: "Belmond-style" scroll transformations.

## Tech Stack
- **Framework**: Next.js 15
- **Styling**: Tailwind CSS 4
- **Animations**: Framer Motion
- **Language**: TypeScript

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
